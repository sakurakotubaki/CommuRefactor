Commu-Refactor (コミュ・リファクタ)
エンジニアのコミュニケーションを「リファクタリング」し、現場での信頼を最大化するためのSwiftUI学習アプリ。

🚀 開発の背景
技術スキルが高いエンジニアであっても、現場での「報告・連絡・相談」の癖（結論が後回しになる、目的語が抜ける等）によって、正当な評価を得られないケースがあります。 本アプリは、日常的なチャットコミュニケーションを**「結論・背景・ネクストアクション」**の構造に再構築（リファクタ）する習慣を身につけるために開発されました。

✨ 主な機能
Situation-Based Learning: 「アカウント発行依頼」「進捗遅延の報告」など、現場で頻出するシチュエーション別のトレーニング。

Before / After Comparison:

Before: 背景から始まり、主語・目的語が曖昧な文章（コミュニケーションコストが高い状態）。

After: 結論から始まり、構造化された文章（信頼を得られる状態）。

Self-Rewrite Mode: 提示されたシチュエーションに対し、自分でリライトを行い、正解例と比較する機能。

🛠 技術スタック
Language: Swift

Framework: SwiftUI

Architecture: MVVM

🎯 ターゲットユーザー
現場でのコミュニケーションにおいて「結局、何が言いたいの？」と言われたことがある方。

報告や依頼の文面を作るのに時間がかかってしまう方。

技術スキルに加えて「伝え方のスキル」を磨き、現場での単価や評価を上げたい方。

📝 5つの黄金ルール（アプリ内ロジック）
本アプリでは、以下の構成での文章作成を推奨しています。

結論 (Conclusion): 何をしてほしいか、何が決まったか。

背景 (Background): なぜそれが必要になったのか。

詳細 (Details): 目的語を明確にした具体的な対象。

次の方針 (Next Action): 自分がどう動くか。

相手への依頼 (Request): 相手に期待するアクション。
